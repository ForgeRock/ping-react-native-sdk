    Enumeration CacheStrategy

    Cache strategies for Android storage operations.

    Defines how the SDK handles caching behavior when storing and retrieving data. These strategies are Android-specific and control how data is cached in memory versus persistent storage.

    Example

    // Use CACHE_ON_FAILURE for resilient storage
    const config = {
      android: {
        keyAlias: 'myKey',
        cacheStrategy: CacheStrategy.CACHE_ON_FAILURE
      }
    };

    // Use NO_CACHE for sensitive data
    const secureConfig = {
      android: {
        keyAlias: 'tokenKey',
        cacheStrategy: CacheStrategy.NO_CACHE
      }
    };

    // Use CACHE for frequently accessed data
    const fastConfig = {
      android: {
        keyAlias: 'sessionKey',
        cacheStrategy: CacheStrategy.CACHE
      }
    };
